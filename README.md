    Windows 用户级安装：C:\Users\<用户名>\.vscode\extensions 全局安装（管理员权限）：C:\Program Files\Microsoft VS Code\resources\app\extensions

    macOS 用户级安装：/Users/<用户名>/.vscode/extensions 全局安装：/Applications/Visual Studio Code.app/Contents/Resources/app/extensions

    Linux 用户级安装：~/.vscode/extensions（如 /home/<用户名>/.vscode/extensions）全局安装（包管理器安装）：/usr/share/code/resources/app/extensions
